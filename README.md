# 更新日志

#### Abnormal-Tools v1.5.0

添加了功能：
1. 其他功能 -> 计分板：记录比赛比分，可以重置
2. 文件功能 -> ncmdump：添加了对其他加密音频文件的支持，基于 [um](https://git.um-react.app/um/cli) 二次开发

其他：
1. 发现了“阶乘计算”输入的数字大于32时会导致输出结果超过 int 范围，现已用 long 代替
2. 将“文件下载器”移到了“文件功能”分区
3. 将“ncmdump”改为了“音乐解锁”
4. 现在的版本号命名遵循 [Semantic Versioning](https://semver.org/) 规范

# 食用方法

电脑安装JRE 17及以上环境并设置了系统变量，下载 exe 或 jar 即可食用

初次启动需要下载约250MB的依赖包，请保持网络畅通
